## Project Overview
This project was completed as part of the **PwC Switzerland Power BI Virtual Job Simulation** on the **Forage** platform.  
The simulation replicates a real-world consulting engagement where HR data is analyzed and transformed into meaningful insights using data visualization techniques.

The primary focus of the project is to evaluate **gender diversity and workforce composition**, with special emphasis on representation at the **executive and senior management levels**.

---

## Objectives
- Analyze HR data to identify diversity and inclusion gaps  
- Assess gender balance across different organizational levels  
- Present insights in a clear, executive-friendly dashboard format  
- Simulate client-facing communication through concise insights  

---

## Tools & Technologies
- Power BI  
- DAX  
- Excel / CSV  

---

## Key Work Done
- **Data Visualization:** Created interactive Power BI dashboards to present key HR KPIs clearly and effectively.  
- **Analytical Problem-Solving:** Analyzed HR data to identify gender balance issues, particularly at the executive level.  
- **Client Communication:** Delivered insights and observations through concise, professional email-style summaries, simulating communication with engagement partners.

---

## Key Outputs
- HR analytics dashboards for senior management review  
- Gender diversity and workforce distribution KPIs  
- Visual insights supporting data-driven decision-making
